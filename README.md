# ひだまり健康チェック管理システム Ver3.9

## 変更点
- 初回ログイン時のパスワード変更を必須化
- 初期アカウント（kanri/staff）が既定パスワードの場合は通常画面に進めない
- 新規職員ID作成時に「初回変更必須」を設定可能
- 管理者による次回ログイン時変更要求に対応
- パスワード最低条件：8文字以上、英字＋数字、ログインIDを含めない、推測されやすい語を禁止

## 起動ファイル
Streamlit Cloud の main file は app.py のままです。


## Ver4.0 利用者ID移行準備版

- 利用者マスタに user_id を追加
- 既存利用者名から user_id を自動生成
- 健康チェック・排泄・短期目標・モニタリング・LIFE ADL に user_id 列を追加
- 画面表示は従来通り利用者名、内部連携は user_id を補完
- 管理者メニューに「利用者ID移行チェック」を追加
- 表記ゆれ候補と主要テーブルの移行見込みを確認可能
